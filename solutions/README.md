# Solutions

Use this directory for policy files and solver variants.

- `public/`: extracted public submissions used as evidence or baselines.
- `ours/`: local solution attempts, ablations, and experimental variants.

Each solution file should include a short header with:

- source or experiment note
- expected score, if known
- simulator command used
- important caveats

Do not edit public baselines in place. Copy them into `ours/` before modifying.
