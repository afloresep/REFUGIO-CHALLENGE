# Experiments

Use this directory for reproducible notes tied to one hypothesis, command, and result.

Recommended filename format:

```text
YYYY-MM-DD-short-title.md
```

Template:

```md
# Experiment: short-title

Date:
Code:
Input:
Hypothesis:
Command:
Result:
Interpretation:
Next:
```

Keep raw output short. Link to generated artifacts or scripts when the output is large.
